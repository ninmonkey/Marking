# Config Paths

## UserDataDir:

- `%APPDATA%\Code`
- `$HOME/.config/Code`

## ExtensionsDir:

- `%USERPROFILE%\.vscode\extensions`
- `~/.vscode/extensions`

## User Settings Dir
   
- `%AppData%\Code\User`
